I believe in open science and reproducible research. Below are the open-source software interfaces and post-processing models developed during my research projects.

🔗

Pangu-Weather DART Model Interface

An ensemble data assimilation interface connecting the machine learning weather prediction model Pangu-Weather with NCAR's Data Assimilation Research Testbed (DART). This code is officially merged into the main NCAR DART codebase.

DART Core Pangu-Weather Ensemble DA Fortran & Python
[GitHub Code Repository] [NCAR DART Project]
🔗

WxChallenge MOS Calibration Model

Statistical post-processing codes using 10-year historical observation data to calibrate and correct the forecast tails of GFS Model Output Statistics (MOS). Used for the national WxChallenge forecasting competition.

WxChallenge MOS Calibration Python GFS Post-processing
[GitHub Profile] [WxChallenge Site]
🔗

Dropsonde Helicity Boundary Layer Composites

Data analysis scripts to process GPS dropsonde composite datasets, calculate boundary layer helicity, and determine the kinematic boundary layer heights of tropical cyclones of varying intensities.
